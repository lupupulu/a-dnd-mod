## 1.遵守事项

1. 尊重随机，无论是dm还是pl，在游玩时请尊重随机性，修改投掷结果一般是不被允许的。
2. 尊重事实，在游玩时，若玩家提出的要求在自然科学中具有可行性，则dm不应拒绝。
3. 尊重死亡，在游玩时，请尊重角色的死亡。
4. 冒险小队一般不参与历史上的重大事件，但他们可能在部分地区十分有名，若需要让他们成为历史上的名人，则推荐从本模组中签出其他分支。
